# Dataset Description

This directory contains information about the dataset used for training the **Diabetic Retinopathy Detection model**.

The dataset consists of retinal fundus images categorized into different severity levels of diabetic retinopathy. These images are used to train a deep learning computer vision model for multi-class classification.

Due to its size, the dataset is **not stored directly in this repository** and must be downloaded separately.

---

# Dataset Overview

The dataset contains **retinal fundus images** representing different stages of diabetic retinopathy. These images are used to train and evaluate deep learning models capable of detecting disease severity from retinal photographs.

The images are organized into **five class folders**, each representing a specific stage of diabetic retinopathy.

---

# Class Distribution

| Class | Description | Number of Images |
|------|-------------|------------------|
| No_DR | No diabetic retinopathy detected | 1805 |
| Mild | Early signs of diabetic retinopathy | 370 |
| Moderate | Moderate retinal damage | 999 |
| Severe | Advanced retinal damage | 193 |
| Proliferative_DR | Most severe stage with abnormal blood vessel growth | 295 |

**Total Images:** 3662
